# Web automation bot
>In this project I crated a program using python selenium that can make signup(outlock,instgram),sign in(instgram,outlock)and like >image (Instagram) (it has limits). 
## Fetchers
>What can it do :
>* **Signup** to *Outlock* 
>* **Signin** to *Outlock*
>* **Singup** to *Instagram*
>* **Singin** to *Instagram*
